Slurm Power Management Guide

Slurm provied an integrated power management system for power capping. This is distict from Slurm's ability to power down idle nodes.

Configuration

The following configuration parameters are available:

  • PowerParameters: Defines power management behavior. Currently valid options are:
    • ClusterCap=#[KW|MW] - Used to establish cluster-wide power cap. Default units are watts.
  • PowerPlugin: Identifies the plugin used to manage system power consumption. Currently valid options are:
    • cray - Used for Cray systems with power monitoring and management functionality included as part of System Management Workstation (SMW) 7.0.UP03.
    • none - No power management support. The default value.
